## Project Overview

Urban Style Clothing is a fictional clothing business based in Cape Town.

The website was created to promote affordable streetwear for teenagers and young adults.

The website provides information about the business, displays products, allows users to send enquiries and gives contact details.

---

## Goals and Objectives

The goals of this project are:

Create a professional online presence

Display products clearly

Allow customers to browse clothing online

Allow customers to contact the business

Create a responsive website

Improve brand awareness
